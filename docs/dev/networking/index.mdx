# Networking
Networking in the programming context, technically, includes both a computer's online *and* offline communication.

<details>
    <summary>What's the difference?</summary>

    To a computer, an offline and online address have the exact same structure. There are special [reserved addresses](ip-address#reserved-address) for local networks; however, apart from that, there is not much of a difference!

</details>
